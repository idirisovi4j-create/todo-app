# Todo App

Простое приложение для задач на JavaScript.

##  Возможности
- Добавление задач
- Удаление задач
- Отметка выполненных (checkbox)
- Сохранение в localStorage
- Красивый UI

##  Технологии
- HTML
- CSS
- JavaScript

##  Описание
Это мой проект для портфолио. Здесь я практиковал DOM, события и работу с localStorage.
